# Report Templates

## Weekly content report

```
CONTENT REPORT — Week of {date}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

SUMMARY
• Published: {n} pieces ({blog: n, social: n, newsletter: n})
• Total reach: {n} ({+/-n%} vs last week)
• Top performer: {title} — {metric}
• Key learning: {insight}

METRICS
                This week    Last week    Change
Organic traffic    {n}          {n}       {+/-n%}
Social reach       {n}          {n}       {+/-n%}
Newsletter opens   {n}%         {n}%      {+/-n%}
Engagement rate    {n}%         {n}%      {+/-n%}

TOP CONTENT
1. {title} — {channel} — {key metric}
2. {title} — {channel} — {key metric}
3. {title} — {channel} — {key metric}

NEXT WEEK
• {planned content 1}
• {planned content 2}
• {focus area}
```

## Monthly content report

```
CONTENT REPORT — {Month Year}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

EXECUTIVE SUMMARY
{3 bullet points on the month's content performance}

KPI DASHBOARD
                  Actual    Target    Status
Published pieces    {n}       {n}     {on/off track}
Organic traffic     {n}       {n}     {+/-n%}
Social followers    {n}       {n}     {+/-n}
Newsletter subs     {n}       {n}     {+/-n}
Engagement rate     {n}%      {n}%    {on/off track}
Content leads       {n}       {n}     {+/-n}

BY CHANNEL
{breakdown per channel with key metrics}

BY PILLAR
{breakdown per content pillar with volume and performance}

TOP PERFORMING
{top 5 pieces with metrics and analysis of why they worked}

SEO PROGRESS
{keyword rankings, new positions, lost positions}

LEARNINGS
{key insights from the month}

NEXT MONTH PRIORITIES
{top 3-5 priorities}
```

## Quarterly content report

Same structure as monthly but with:
- Quarter-over-quarter trends
- Strategic alignment assessment
- Resource allocation review
- Budget vs actual
- Competitive positioning update
