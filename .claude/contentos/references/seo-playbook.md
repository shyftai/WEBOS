# SEO Playbook

## Core principles

1. **Search intent is king** — match content type to what the searcher actually wants
2. **E-E-A-T matters** — Experience, Expertise, Authoritativeness, Trustworthiness
3. **Technical foundation first** — no amount of content fixes a broken site
4. **Topic clusters > individual pages** — build authority on topics, not just keywords
5. **User signals drive rankings** — time on page, bounce rate, pogo-sticking all matter

## Search intent mapping

| Intent | Searcher wants | Content type | Example |
|--------|---------------|-------------|---------|
| Informational | Learn something | Blog, guide, how-to | "what is content marketing" |
| Navigational | Find a specific page | Landing page, homepage | "hubspot blog" |
| Commercial | Compare options | Comparison, review, listicle | "best SEO tools 2026" |
| Transactional | Take action | Product page, pricing, signup | "ahrefs pricing" |

## On-page SEO checklist

- [ ] Meta title: ≤60 chars, includes primary keyword, compelling
- [ ] Meta description: ≤155 chars, includes keyword, has CTA
- [ ] H1: single, includes primary keyword, matches search intent
- [ ] Heading hierarchy: H1 → H2 → H3 (no skipping levels)
- [ ] Primary keyword: in first 100 words, naturally placed
- [ ] Internal links: 3-5 relevant links to other content
- [ ] External links: 2-3 authoritative sources
- [ ] Image alt text: descriptive, includes keyword where natural
- [ ] URL: short, descriptive, includes keyword
- [ ] Schema markup: appropriate for content type
- [ ] Word count: competitive with top-ranking pages

## Topic cluster strategy

```
         [PILLAR PAGE]
         (broad topic, high volume)
              │
    ┌─────────┼─────────┐
    │         │         │
[CLUSTER]  [CLUSTER]  [CLUSTER]
(subtopic) (subtopic) (subtopic)
    │         │         │
[SUPPORT] [SUPPORT] [SUPPORT]
(long-tail) (long-tail) (long-tail)
```

- Pillar page: comprehensive overview (2,000-4,000 words)
- Cluster pages: deep dives on subtopics (1,500-2,500 words)
- Support pages: specific long-tail queries (800-1,500 words)
- All pages link to pillar. Pillar links to all clusters.

## Content refresh strategy

- Check rankings monthly for position drops
- Refresh content older than 6 months that ranks positions 5-20
- Add new data, examples, sections as needed
- Update publish date after significant refreshes
- Don't touch content ranking #1-3 unless it's dropping

## Technical SEO basics

- Site speed: Core Web Vitals passing
- Mobile-first: responsive design, mobile-friendly content
- HTTPS: required, no mixed content
- XML sitemap: submitted to Search Console
- Robots.txt: not blocking important pages
- Canonical tags: set correctly, no conflicts
- 404 handling: custom page, redirect broken pages
- Schema: Article, FAQ, HowTo, Breadcrumb as appropriate
