# Changelog

## v1.0.0 — Launch

### Core system
- Boot sequence with ASCII banner and system status
- Workspace architecture with sources of truth
- Role-based onboarding (Content Creator, Content Strategist, Social Media Manager, SEO Specialist, Head of Content, Agency)
- MCP server and API key detection
- Solo/team collaboration mode
- Quality gates (7-point review)
- Learnings system (persistent intelligence)
- Content roadmap with auto-generated to-dos

### Commands (25+)
- Daily: today, dashboard
- Setup: onboard, research
- Create: brief, write, repurpose, batch
- SEO: seo-audit, keywords, optimize, cluster
- Publish: publish, calendar
- Social: social, community
- Intel: trends, competitor
- Measure: analytics, report
- Review: review, debrief
- Newsletter: newsletter
- Agency: portfolio

### Integrations
- SEO: Ahrefs, SEMrush, Moz, Search Console, Google Analytics
- Publishing: WordPress, Webflow, Ghost
- Social: LinkedIn, Twitter/X, Instagram, TikTok
- Scheduling: Buffer, Hootsuite
- Newsletter: Beehiiv, ConvertKit, Substack
- Research: Exa, Firecrawl
- Monitoring: Mention, Brand24
- Community: Reddit, Discord
- Design: Canva
- Scale: Claude API for batch generation
- Collaboration: Supabase for team mode
